# Carusel
AVISO: Proyecto creado con soporte de la IA y/o colaboradores a cargo del desarrallador principal Eric Munive Garcia

**Carusel** es una innovadora aplicación de escritorio diseñada para gestionar el portapapeles de manera visual e intuitiva. Este proyecto ha sido **creado enteramente por Eric Munive Garcia** y forma parte de un **portfolio para las empresas**.

## 🚀 ¿Qué hace el proyecto?

Carusel actúa como un gestor avanzado de portapapeles (Clipboard Manager) que va un paso más allá de las herramientas tradicionales. La aplicación registra y guarda en memoria temporal los diferentes elementos que el usuario copia en su sistema y los organiza en una interfaz dinámica, en forma de "rueda" o "carrusel", para un acceso rápido y fluido.

### 📋 Características principales:

- **Soporte multiformato:** Permite guardar y gestionar no solo texto, sino también imágenes, vídeos y documentos copiados.
- **Interfaz en Carrusel:** Acceso visual a los elementos del portapapeles en un diseño de rueda rotatoria mediante JavaFX, mejorando la eficiencia y la productividad.
- **Previsualización de contenido:** Genera y muestra vistas previas (thumbnails) del contenido almacenado, especialmente útil para imágenes y archivos multimedia.
- **Atajos de teclado:** Integración con atajos de teclado para la apertura y cierre rápido de la interfaz en forma de rueda, así como el uso de la rueda del ratón para navegar entre los distintos elementos almacenados.
- **Control en segundo plano:** La aplicación se ejecuta de manera discreta en el sistema, siempre lista para capturar y ofrecer el contenido copiado cuando sea necesario.

## 🛠️ Tecnologías

Este proyecto está desarrollado principalmente en:
- **Java**: Lenguaje de programación base.
- **JavaFX**: Para la construcción de la interfaz gráfica y las complejas animaciones del carrusel.
- **ClipboardFX**: Biblioteca personalizada para la interacción a bajo nivel con el portapapeles del sistema operativo.
